Exemplo 11 HTML - Formulários

Um formulário em HTML é uma estrutura de coleta de dados que utiliza tags específicas como, form, label, input, button, fieldset e legend para descrever a função e o significado de cada campo. Diferente de usar apenas divisores genéricos para fins estéticos, a semântica garante que navegadores, leitores de tela e motores de busca compreendam a relação lógica entre os dados solicitados e seus rótulos, aumentando a acessibilidade e a eficiência no processamento da informação.

Os métodos GET e POST são as formas mais comuns de enviar dados de um formulário HTML para um servidor, definidos pelo atributo method na tag form.

  • GET: Envia os dados anexados à URL (query string), visíveis para o usuário, ideal para consultas e buscas.
  • POST: Envia os dados no corpo da requisição (body), não aparecendo na URL, sendo o método recomendado para dados sensíveis, login ou modificação de dados no servidor.
