Exemplo 18 HTML CSS - Seletor de Classe

Seletor de classe

O seletor de classe CSS é uma fomra de aplicar estilos a elementos HTML especificos, identificados pelo atributo class, oque permite que a regra possa ser reaproveitada em varios elementos HTML.

Neste exemplo estamos usando vários seletores de classe. cada seletor de classe começará sempre com um ponto (.) seguido do nome do seletor{propriedade: valor}
