Exemplo 8 - Principais Elementos HTML - Imagens

O elemento HTML img representa a inserção de imagens no documento HTML, permitindo também o uso da tag figure e figcaption que começaram a ser utilizadas a partir da versão 5 do html - html5.

Exemplo de uso da tag img

Para usar a tag img deve ser informado o caminho da imagem. Este caminho deve ser informado no atributo src.

Outro atributo também utilizado da tag img é o atributo alt. Esse atributo será utilizado para mostrar um texto alternativo quando a imagem não for mostrado.
