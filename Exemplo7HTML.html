Exemplo 7 - HTML Listas Ordenadas e Não Ordenadas

Este exemplo mostra como criar listas ordenadas e não ordenadas incluindo subitens(listas aninhadas).

Descrição das tags utilizadas na criação das listas.

  • ol:Define uma lista ordenada(com números, letras e ou algarismos romanos)
  • ul: Define uma lista não ordenada ( com marcadores com pontos e quadrados)
  • li: Define um elemento na lista

Exemplo de lista Ordenada <ol>

  1. Item 1
    1. Subitem 1.1
    2. Subitem 1.2
    3. Subitem 1.3
    4. Subitem 1.4
  2. Item 2
  3. Item 3
  4. Item 4
    1. Subitem 4.1
    2. Subitem 4.2
    3. Subitem 4.3
    4. Subitem 4.4
      1. Subitem 4.4.1
      2. Subitem 4.4.2
      3. Subitem 4.4.3
  5. Item 5

Listas Não-Ordenadas <ul>

  • Item 1
  • Item 2
    • Subitem 2.1
    • Subitem 2.2
    • Subitem 2.3
      • Subitem 2.3.1
      • Subitem 2.3.2
      • Subitem 2.3.3
        • Subitem 2.3.3.1
        • Subitem 2.3.3.2
          • Subitem
          • Subitem
      • Subitem 2.3.4
    • Subitem 2.4
  • Item 3
  • Item 4
  • Item 5
