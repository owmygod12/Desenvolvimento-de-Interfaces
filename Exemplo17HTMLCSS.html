CSS - cascsding Style Sheets

oque é css

O CSS - folhas de estilo em cascata é uma tecnologia usada para definir a aparencia visual de um documento em HTML. enquanto o HTML possui tags que estruturam o documento HTML, o CSS diz como esse conteudo deverá ser exibido

oque sao regras CSS

toda instrução de estilo em CSS é chamada de regra, sendo que uma regra é composta por:

  • seletor: indica quais elementos HITML serão estilizados e podem ser de 3 tipo: Id, Tipo e Classe
  • bloco de declarações: não é nada mais que um par de propriedade: valor
  • propriedade: é o aspecto visual que deverá ser controlado, tais como: cor de fonte, margem , cor de fundo, tipo de fonte e etc
  • valor é o atributo e o atributo da propriedade que queremos modificar

seletor de tipo

o seletor de tipo seleciona todos os elementos de uma determinada tag HTML na pagina. A sintaxe desse seletor é o nome da tag HTML, sem nenhum prefixo. neste exemplo estamos usando varios seletores de Tipo
