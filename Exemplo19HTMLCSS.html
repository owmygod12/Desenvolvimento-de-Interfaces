Seletor de ID

O seletor de ID seleciona um único elemento identificado pelo atributo id. A sintaxe usa uma cerquilha (#) antes do nome do id. #cabecalho { font-style: italic; font-size: 1.5rem; }

Regra fundamental: um id deve ser único na página — não pode existir dois elementos com o mesmo id.
