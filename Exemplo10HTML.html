    • conceitos gerais

    A Web Semântica é uma extensão da Web atual, idealizada pelo Consórcio World Wide Web (W3C), onde as informações ganham significado bem definido, permitindo que máquinas processem, interpretem e conectem dados de forma inteligente. Ela transforma documentos em "dados conectados", facilitando buscas automáticas, interoperabilidade e compreensão do contexto por agentes de software.

    características

    • Conteúdo Compreensível por Máquinas (Machine-Readable): Os dados são estruturados com metadados que permitem que agentes de software entendam o contexto e o significado do conteúdo, superando a limitação atual onde máquinas apenas "enxergam" texto sem contexto.
    • Uso de Metadados e Ontologias: Utiliza descritores adicionais (metadados) para definir o conteúdo. Ontologias e taxonomias são empregadas para definir relações entre conceitos, organizando o conhecimento.
    • Interoperabilidade e Integração de Dados: A Web Semântica facilita a conexão de dados de diferentes fontes, permitindo que sistemas heterogêneos compartilhem e reutilizem informações de maneira eficiente.
    • Dados Conectados (Linked Data): A estrutura baseia-se em "triplas" (sujeito, predicado, objeto), seguindo o modelo RDF (Resource Description Framework), que conecta informações através da web, criando uma rede de conhecimento.
    • Melhor Recuperação de Informação: Torna as buscas mais precisas. Em vez de retornar apenas links baseados em palavras-chave, a Web Semântica fornece respostas diretas ao entender o que o usuário está procurando.
    • Automação de Agentes de Software: Permite que programas de computador (agentes) realizem tarefas complexas automaticamente, como integração de dados e inferência de novos conhecimentos a partir de dados existentes.
    • Padrões W3C: É baseada em padrões definidos pelo World Wide Web Consortium (W3C), incluindo RDF, OWL (Web Ontology Language) e SPARQL (linguagem de consulta para RDF).

    Aplicações

    • Busca e Recuperação Inteligente: Motores de busca compreendem o contexto da consulta, não apenas palavras-chave, fornecendo respostas diretas (ex: Google Knowledge Graph).
    • Assistentes Virtuais e Bots: Ferramentas como Siri e Alexa utilizam dados estruturados para entender e responder a perguntas complexas.
    • Dados Conectados (Linked Data): Conecta dados de diferentes fontes para análise e reutilização, comum em áreas como ciência, saúde e governos.
    • Comércio Eletrônico e Comparação de Preços: Agentes automáticos comparam produtos de diversas lojas analisando características, preços e compatibilidade.
    • Organização de Conhecimento: Uso de vocabulários (SKOS) e ontologias (OWL) para estruturar grandes bases de dados, facilitando a navegação e interoperabilidade entre sistemas.
